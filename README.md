# Samadhi Senanayake - Personal Academic Website

This is the source code for my personal academic website, hosted on GitHub Pages using Jekyll.

## About

This website showcases my research in lamprey adaptive immunity, variable lymphocyte receptors, and infectious disease diagnostics.

## Setup

To run this site locally:

1. Install Jekyll and Bundler:
   ```bash
   gem install jekyll bundler
   ```

2. Clone this repository:
   ```bash
   git clone https://github.com/samadhisenanayake/samadhisenanayake.github.io.git
   cd samadhisenanayake.github.io
   ```

3. Serve the site:
   ```bash
   jekyll serve
   ```

4. Visit `http://localhost:4000` in your browser

## Customization

- Add your profile photo to `assets/img/profile.jpg`
- Update your CV PDF at `assets/CV_Samadhi_Senanayake.pdf`
- Edit `_data/social.yml` to update contact links
- Add new publications to `_data/publications.yml`
